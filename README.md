# Segjong-Dormitory

**Segjong-Dormitory**는 세종대학교 기숙사 생활을 편리하게 만들어주는 모바일 앱입니다. 현재 로그인, 공지사항, 민원 접수, 마이페이지, 식단 조회 UI가 구현되어 있습니다.
